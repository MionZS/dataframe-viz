# GCP Visualizer TUIs

This is an isolated, standalone project containing CSV and Parquet visualization and inspection tools.

**Extract this ZIP file and see QUICK_START.md for instructions.**

## Contents

- `visualizers/` - TUI tools (LazyFrame Viewer, Data Inspector)
- `requirements.txt` - Python dependencies
- `README.md` - Full documentation
- `SETUP.md` - Setup and troubleshooting
- `examples/` - Helper scripts for Windows and Linux/Mac

## Getting Started (30 seconds)

1. Extract ZIP file
2. Open terminal in the extracted folder
3. Create virtual environment: `uv venv`
4. Activate: `.venv\Scripts\activate` (Windows) or `source .venv/bin/activate` (Linux/Mac)
5. Install: `uv pip install -r requirements.txt`
6. Run: `python visualizers/lazy_frame_viewer.py --file YOUR_FILE.csv` (or .parquet)

For detailed instructions, see **SETUP.md**
